『 **アッチむいて恋** 』は  2010年  4月30日  に  ASa Project  から発売された  18禁  恋愛アドベンチャーゲーム
。男として学園に通い、女として女子寮で暮らす主人公（  男の娘  ）の二重生活を描いたASa Projectの第3作目。「アチ恋」と略される。

##  あらすじ



田舎生活に憧れる主人公・鳴海浩介は、努力の甲斐あって田舎の学園へ転入することになった  [  1  ]
。だが学園側の手違いにより寮が空いておらず、他の物件も見つからない  [  1  ]  。途方にくれる浩介は、学園側から女子寮への入居を命じられる  [
1  ]  。かくして学園では男、寮では女装して女という二重生活を送ることになる  [  1  ]  。

##  登場人物



ソフマップ秋葉原・アミューズメント館で行われた発売記念イベント  （2010年4月30日撮影）

###  メインキャラクター



鳴海 浩介（なるみ こうすけ）/干支名 真恵（えとな まえ）

     身長：165cm、  血液型  ：A型  [  2  ] 
     この物語の主人公  [  1  ]  。春咲学園の2年生。訳あって、学園では男、女子寮では「干支名 真恵」としての二重生活をする事になった  [  1  ]  。 
成沢 美奈子（なるさわ みなこ）

     声：  松田理沙 
     身長：158cm、3サイズ：93/60/88、誕生日：1月27日、  血液型  ：A型 
     春咲学園の2年生。浩介のクラスメイトで、朱とは親友。寮メンバーの中で家事能力に優れており  [  1  ]  、特に食事に関してはほぼ一手に引き受けている。 
     その一方で、性に関する知識が欠如しており、ある単語の意味を知らずに連呼して、意味を知ったその後の三日間は食事の献立は悲惨なものとなった。 
織葉 朱（おりば あか）

     声：  五行なずな 
     身長：160cm、3サイズ：89/58/87、誕生日：10月10日、  血液型  ：B型 
     春咲学園の2年生。浩介のクラスメイト。親友の美奈子とは対照的に家事や炊事は全く出来ないが、勉強の成績は良い  [  3  ]  。都会の男に対する偏見から浩介の事を毛嫌いしている  [  3  ]  。一方、女子寮では素直な一面を見せる  [  3  ]  。 
八乙女 優由（やおとめ ゆうゆ）

     声：  遠野そよぎ 
     身長：153cm、3サイズ：81/56/80、誕生日：4月1日、  血液型  ：B型 
     春咲学園の1年生。チアリーダー部に所属している  [  1  ]  。運動神経抜群。寮のムードメーカー兼トラブルメーカー  [  1  ]  。 
     学校の廊下で浩介にぶつかり一目惚れをして以来、浩介に猛烈なアプローチを仕掛けてくる。 
柊 かぐや（ひいらぎ かぐや）

     声：  春河あかり 
     身長：161cm、3サイズ：86/56/86、誕生日：11月27日、  血液型  ：O型 
     春咲学園の3年生。被服部の部長を務めており、被服部にある洋服の大半を作ってきたほか、デザイナーを目指している  [  3  ]  。自らの好みや理想を追求する一方、興味のないものには全く関心を示さない  [  3  ]  。 
     物語中盤で、浩介の女装のことを知るが、交換条件を持ちかけ、黙っていることを承諾する。 
鳴海 ルナ（なるみ るな）

     声：  北都南 
     身長：150cm、3サイズ：78/55/80、誕生日：7月18日、  血液型  ：A型 
     主人公の実妹で、通信制学園の一年生。兄を寮の女子に取られないよう見張る目的で入寮。兄の女装時の名前に合わせ、干支名ルナと名乗る。 
     しっかりした性格をしている反面、褒められるのに弱い  [  1  ]  。 

###  サブキャラクター



千歳 杏樹（ちとせ あんじゅ）

     声：  木村あやか  [  2  ] 
     春咲学園の3年生。浩介と同じ寮に住む「女装の先輩」で、浩介と違い学園でも女装をしている  [  3  ]  。 
三津原 鈴（みとはら りん）

     声：  杉原茉莉 
     春咲第三女子寮の寮長で、口が悪い  [  3  ]  。寮生達からは「お鈴さん」と呼ばれている。酒  [  3  ]  、タバコ  [  3  ]  、パチンコ、金をこよなく愛する。 
春咲 むつみ（はるさき むつみ）

     声：  貴坂理緒 
     春咲学園の理事長  [  3  ]  。三津原鈴とは昔からの幼馴染。浩介の二重生活をサポートする立場だがあまり役には立っておらず、むしろ妙な言動で振り回している。また、都合が悪くなると笑ってごまかす  [  3  ]  。 
風間 慎吾（かざま しんご）

     声：  島田友樹 
     春咲学園の2年生。浩介・美奈子・朱のクラスメイト。同性愛紛いの言動で気味悪がられており、男友達は浩介ぐらいしかいない。 
大華 未虎（おおか みとら）

     声：  奥山歩 
     春咲学園の2年生。被服部に所属。担当はアシスタント兼、写真撮影。 
明方 メテオ（あけがた めてお）

     声：  藤原鞠菜 
     春咲学園の1年生で、優由の親友にあたる。チアリーダー部に所属。無口なタイプ。以前は引っ込み思案で体も弱かったためいじめにあっていたが、優由の何気ない一言によって救われた。その後は優由を親友として慕っている。髪の色は明るい緑。弁当の仕切りなどに用いる  ばらん  をヒントにキャラクターデザインがなされた  [  4  ]  。

##  反響



『  女装少年ゲーム大全：2009-2011  』（  ミリオン出版
）が本作の干支名真恵と千歳杏樹を「男の娘」として取り上げている。主人公となる「男の娘」にくわえ、もう一人先輩の「男の娘」が登場する点を、女装潜入ものとしては珍しいと評している。本作は、主人公が男のときと女のときとでヒロインたちの反応が変わるところがポイントだとしている
[  2  ]  。
